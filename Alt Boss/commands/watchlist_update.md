# Команда /watchlist update

Пересмотр всех токенов в watchlist.

## Протокол

1. Прочитай `watchlist/watchlist.yaml`
2. Для каждого токена:
   - Получи текущую цену и метрики
   - Поищи свежие новости (WebSearch)
   - Проверь: выполнились ли условия для входа?
   - Запусти сокращённую дискуссию (5 раундов) с учётом новой информации
3. Обнови watchlist: статус, цену, дату
4. Выведи сводную таблицу изменений

## Важно
Watchlist ≠ мусорка. Только токены с конкретным триггером для пересмотра.
